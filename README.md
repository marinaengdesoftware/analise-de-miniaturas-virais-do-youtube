# Análise de Padrões de Miniaturas Virais do YouTube

Este projeto apresenta um estudo de engenharia reversa aplicado a **miniaturas (thumbnails) e títulos de vídeos virais no YouTube**.

O objetivo é identificar **estruturas visuais, emocionais e narrativas recorrentes** utilizadas em conteúdos de alto desempenho e transformar essas estruturas em **templates replicáveis de criação de thumbnails**.

---

# Objetivo do Estudo

Investigar como determinados padrões visuais influenciam:

- taxa de clique (CTR)
- retenção inicial
- percepção de autoridade
- curiosidade e tensão narrativa

A partir dessa análise, foi possível transformar esses padrões em **templates visuais reutilizáveis para criação de conteúdo**.

---

# Canal Utilizado no Estudo

Canal analisado:

**Histórias da Bíblia**

Características observadas:

- thumbnails cinematográficas
- narrativa épica religiosa
- uso de arquétipos visuais fortes
- personagens bíblicos com autoridade visual
- cenários dramáticos (fogo, batalha, ruínas, céu épico)

---

# Metodologia

O estudo foi conduzido em três etapas principais.

### 1️⃣ Coleta de dados

Foram coletadas miniaturas de vídeos com alto desempenho dentro do canal analisado.

Elementos analisados:

- composição visual
- paleta de cores
- símbolos recorrentes
- estrutura narrativa da imagem
- relação entre thumbnail e título

---

### 2️⃣ Extração do DNA visual

Foi aplicado um framework de análise chamado:

**VDNA-TEMPLATE™**

Esse framework permite decompor a thumbnail em múltiplas camadas:

- identidade visual macro
- composição e layout
- textura e acabamento
- símbolos e arquétipos
- gatilhos psicológicos

O resultado dessa etapa é um **modelo estruturado em JSON** representando o DNA visual da thumbnail.

---

### 3️⃣ Modelagem de templates criativos

Após a análise, os padrões foram transformados em **templates visuais replicáveis**, permitindo gerar novas thumbnails mantendo o mesmo padrão emocional e estrutural.

Esses templates incluem:

- regras de composição
- paleta visual dominante
- arquétipos emocionais
- símbolos narrativos
- estrutura de foco visual

# Principais Descobertas

O estudo revelou que miniaturas virais seguem padrões relativamente consistentes.

Entre os principais elementos identificados:

### Estrutura de composição

- rosto dominante ocupando grande parte da imagem
- posicionamento lateral do personagem
- fundo narrativo dramático

### Paleta visual

base:

- tons terrosos
- areia / bege
- marrom

acento:

- fogo laranja
- dourado

### Textura

- rosto extremamente nítido
- fundo com fumaça ou haze
- contraste alto

### Hierarquia visual

1️⃣ personagem principal  
2️⃣ símbolo narrativo  
3️⃣ cenário dramático

---

# Template Emocional

Os arquétipos emocionais dominantes identificados foram:

- autoridade
- ameaça
- proteção
- revelação

Estrutura psicológica predominante:
ameaça visível + figura de autoridade + promessa de proteção


Esse mecanismo cria **curiosidade e urgência**, aumentando a probabilidade de clique.

---

# Aplicação

A partir do template identificado, foi possível gerar novas thumbnails utilizando ferramentas de geração de imagem mantendo o mesmo padrão emocional e estrutural.

O objetivo não é copiar conteúdos existentes, mas **compreender e reproduzir padrões estruturais de comunicação visual**.

---

# Observação

Este projeto tem finalidade educacional e analítica.

Ele não pretende reproduzir integralmente o funcionamento do algoritmo do YouTube, mas sim estudar **padrões visuais e psicológicos utilizados em conteúdos de alto desempenho**.
